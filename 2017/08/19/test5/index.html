    搭建hexo博客

    搭建流程

    第一步： 安装node.js以及本地Hexo初始化
    第二步： 服务器环境搭建，包括安装 Git 、Nginx配置 、创建 git 用户
    第三步： 使用Git自动化部署发布博客

    本地环境

    下载安装node.js

    windows:

    下载安装包,安装完成后 点击左下角开始菜单=》运行=》输入”cmd” => 输入命令”node –version”

    上面显示vx.x.x的版本信息说明安装成功

    Ubuntu/Debian:

    初始化Hexo博客

    首先创建 你的Hexo 目录

    安装 hexo-cli

    初始化该文件夹

    安装hexo的依赖,hexo是基于nodejs开发的，npm是nodejs的包管理工具

    安装两个插件， hexo-deployer-git 和 hexo-server ,这俩插件的作用分别是使用Git自动部署，和本地简单的服务器。

    以及一些其他插件（非必选）

    生成自己的第一篇文章

    使用 hexo new <文章名称> 来新建文章，该命令会成成一个 .md文件放置在 “sources/_posts”文件夹内.
    .md是markdown文件的扩展名，我们可以使用自己喜欢的文本编辑工具去编辑markdown，比如Sublime Text;notepad.

    编辑完毕以后， 使用hexo g将 .md文件渲染成静态文件，然后启动hexo-server

    用浏览器打开 http://localhost:4000 如果看到 hexo 的初始页面证明安装成功。
