    将 Hexo 博客部署到 GitHub

    安装 Git 部署模块

    编辑hexo主目录下的_config.yml

    修改repo地址为自己的github地址

    添加git个人信息

    添加ssh公匙

    复制公匙内容,然后去github网站–>settings–>ssh and gpg key–>new ssh key黏贴进去

    然后hexo三连,把本地文件部署到github上
