    一个好用的图形化you-get

    需要 java 运行环境。jre8 or jdk

    一个基于开源项目you-get的GUI程序，可以运行多种平台上，功能如下：

    下载网站上的视频、音频和图片（支持的网站列表可以在下方找到）
    更多的功能正在开发中

    GUI-YouGet 整合了两款著名的视频网站命令行下载工具 you-get 和 youtube-dl，为 Windows 提供一个更加易用的图形视频下载界面，几乎囊括了主流视频网站，

    下载地址:https://github.com/ingbyr/GUI-YouGet/releases
    使用:
    输入网址,然后点击下载.(某些如youtube当然还需要设置代理才能下载)

    点击列表中想下载的分辨率
