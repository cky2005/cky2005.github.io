    github-pages 自定义二级域名加https

    方法是通过免费不限流量CDN Cloudflare 实现.

    示例,用的是二级域名blog.233666.org绑定github-pages

    当然首先你要有域名,域名dns新增一个项目:

    子域名:blog ,记录类型:CNAME ,ip地址/目标主机填写: 你的github-pages.github.io

    然后去Cloudflare注册一个账号 https://www.cloudflare.com

    加你的网站进去,譬如我加的是233666.org

    去你的域名服务商 将 域名服务器设置为Cloudflare提供的2个域名服务器:

    art.ns.cloudflare.com
    elsa.ns.cloudflare.com
    稍微等一会儿等设置生效.

    回到cloudflare的overview设置页,查看状态是否active

    cloudflare - dns设置页 加一个项目(不用加aaa,www那些,就加下面这个blog就行)

    子域名:blog ,记录类型:CNAME ,ip地址/目标主机填写: 你的github-pages.github.io

    cloudflare - cypto设置页

    ssl 设置为 flexible , Automatic HTTPS Rewrites设置为 on
    (设置后,cloudflare会为你自动申请一个证书,要等上一段时间才能生效)

    cloudflare - Page Rules设置页

    `create page rules’ 填 http:/blog.你的域名.org/*

    注意:设置生效可能要等上一段时间的,不是一设置就立即生效的.

    示例地址: https://blog.233666.org 你可以看看网站的证书,显示的是cloudflare的证书.
