    Fiddler抓包工具

    抓包工具Fiddler详解

    第一步：下载神器Fiddler安装: 下载链接

    第二步：设置Fiddler(配置完后记得要重启Fiddler)

    打开Fiddler,选择菜单

    • Tools-> Fiddler Options ->HTTPS
      选中"Decrpt HTTPS traffic", //Fiddler就可以截获HTTPS请求
      选中"Ignore server certificate errors"
    • Tools-> Fiddler Options ->Connections
      选中"Allow remote computers to connect". //是允许别的机器把HTTP/HTTPS请求发送到Fiddler上来
      记住这个端口号,默认是:8888

    第三步:获取电脑上的ipv4地址

    示例: 获得IPv4 为 192.168.123.12
    第一种方法:进入cmd,输入ipconfig

    第二种方法:鼠标指针移到fiddler界面右上角的工具栏Online小图标会弹出一个信息框

    第四步：设置手机

    确定一下手机和PC是连接在同一个局域网中
    进入手机的设置->进入WLAN/wifi设置->选择连接到的wifi无线网
    选择显示高级选项，将代理设置成手动，将第三步获取到的ipv4地址和第二步获取的端口号填入，点击保存。

    第五步：手机下载Fiddler的安全证书

    手机自带的浏览器打开上面获取到的ipv4:端口(ucweb之类不行,要用系统自带的),
    示例: http://192.168.123.12:8888， 点"FiddlerRoot certificate" 然后安装证书

    • 现在可以在手机上打开某app,然后看看电脑上fiddler是否抓包成功了.
