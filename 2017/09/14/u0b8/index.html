部署免费博客到代码托管空间

一些代码托管空间有提供个人空间可以用来搭建静态博客
Github
Heroku
Bitbucket
码云:
码市Coding

1.网站上注册一个用户名

2.创建一个仓库
示例:用户名”test”(博客名要和用户名一样,譬如用户名test,仓库名字也要为test)
GitHub/码云/码市仓库要设置开启pages功能,bitbucket不用设置,heroku要对app执行heroku ps:scale web=1.

代码托管创建的仓库名
githubtest.github.io
heroku创建一个名字为test的app
bitbuckettest.bitbucket.io
码云test
码市test

3.用的hexo静态博客,配置文件里面的repo:
推荐都增加一个ssh密匙,增加密匙后部署就不用输入密码了

# npm install hexo-deployer-git --save
deploy:
  type: git
  repo: git@git.oschina.net:/test/test.git     #码云
  repo: git@github.com:test/test.github.io.git  #github
  repo: git@git.coding.net:test/test.git  #码市
  repo: git@bitbucket.org:test/test.bitbucket.io.git  #bitbucker

# npm install hexo-deployer-heroku --save
deploy:
  type: heroku
  repo: https://git.heroku.com/test.git  #heroku
